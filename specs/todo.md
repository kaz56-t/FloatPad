# FloatPad — 追加機能アイデア

現状の実装（メモ・電卓・Webビューア・設定パネル）をベースに、
あったら便利そうな機能を優先度別にリストアップする。

---

## 優先度: 高（使用頻度・実現性ともに高い）

### メモ
- [x] **複数メモ（タブ切り替え）**
  タブで切り替え、追加（`+`）・削除（`×`）・名前変更（dblclick + `prompt()`）・起動時に最後のメモを復元。
  ファイル: `memo-{id}.txt`

- [x] **行番号表示 on/off**
  コードメモを書く人向け。設定パネルにトグルを追加。

- [ ] **メモ名の変更 inline 化**
  現在は `prompt()` ダイアログ。タブ上でその場編集（`contenteditable` or `<input>`）に置き換え。

- [ ] **箇条書き・インデント対応**
  `Tab` キーでインデント、`-` や `*` で自動箇条書き、行頭記号をツールバーで挿入。

- [ ] **Markdown プレビュー切り替え**
  ツールバーに「編集 / プレビュー」トグルを追加。
  `marked.js` など軽量ライブラリで HTML レンダリング。

- [ ] **テキスト検索（Ctrl+F）**
  メモ内をインクリメンタル検索。マッチ箇所をハイライト。

- [ ] **フォントサイズ変更**
  設定パネルに `font-size` スライダーを追加（12px〜20px 程度）。
  settings.json に保存して再起動後も維持。

### 電卓
- [ ] **計算履歴をクリップボードにまとめてコピー**
  履歴エリアを右クリック or ボタンで全履歴を一括コピー。

- [ ] **科学計算モード切り替え**
  `sin / cos / tan / log / √` ボタンを追加した拡張モード。
  ボタン `[+科学]` でパネルを展開。

- [ ] **単位変換サブパネル**
  長さ・重さ・温度などをその場で変換。電卓とは別の薄いドロワーとして展開。

### Webビューア
- [ ] **ブックマーク機能**
  ☆ボタンでURLをブックマーク登録。ドロップダウンで一覧表示・ジャンプ。
  bookmarks.json に保存。

- [ ] **検索エンジン選択**
  設定パネルに Google / Bing / DuckDuckGo などを選べるドロップダウンを追加。
  URLでないワードを入力した際の飛び先を変更。

- [ ] **ページ内テキスト選択 → メモに貼り付けボタン**
  webview 上で選択したテキストを「メモに追記」するコンテキストメニュー項目を追加。

---

## 優先度: 中（あると差がつく機能）

### 全体 / ウィンドウ
- [ ] **最小化ボタン**
  タイトルバーに `─` ボタンを追加（`win.minimize()` via IPC）。

- [ ] **グローバルホットキー（システム常駐）**
  `Ctrl+Shift+Space` など OS 全体のショートカットでウィンドウを前面に呼び出す。
  Electron `globalShortcut` API で実装。

- [ ] **ウィンドウサイズのプリセット**
  「小 / 中 / 大」プリセットボタンをタイトルバーに追加。
  ウィジェット的に使うときは「小」、作業中は「大」に素早く切り替え。

- [ ] **ミニモード（折りたたみ）**
  タイトルバーのダブルクリックでパネル部分を隠し、タイトルバーのみの細長いバーに縮小。
  再度ダブルクリックで展開。

- [ ] **スナップ位置記憶**
  画面端や角に寄せたとき「スナップ」する挙動を追加。
  ウィンドウ位置を settings.json に保存する機能は既存なので、
  スナップ座標候補を計算して近づいたらそこに吸い付かせる。

- [ ] **トレイアイコン常駐**
  ✕ ボタンで閉じてもシステムトレイに残り、アイコンクリックで再表示。
  完全終了はトレイメニュー「終了」から。

- [ ] **複数ウィンドウ対応**
  「新しいウィンドウ」メニューで FloatPad を複数起動し、
  それぞれ独立したメモ・タブ状態を持てるようにする。

### メモ
- [ ] **定型文タブ**
  よく使うテキストを登録し、ボタン一押しでクリップボードにコピー。
  定型文の追加 / 編集 / 削除・並び替え（ドラッグ or 上下ボタン）。
  `user-data/snippets.json` に保存。

- [ ] **メモのエクスポート（PDF / HTML）**
  設定パネルや右クリックメニューから現在のメモを PDF や HTML として保存。

- [ ] **ハイライト（蛍光ペン）**
  テキスト選択後にハイライト色を塗れる機能。メモをメモらしく。

### 電卓
- [ ] **式の直接編集**
  現在の入力式をテキストとして直接編集できる入力欄を追加。
  長い計算式の途中修正が容易になる。

- [ ] **為替レート換算**
  設定で API キーを登録（または定期取得）して、
  `×USD` のような後置記法でリアルタイム換算。

### Webビューア
- [ ] **簡易履歴パネル**
  訪問URL履歴をサイドドロワーで表示。クリックで再訪問。
  history.json に保存（件数上限あり）。

- [ ] **ズームイン / アウト**
  `Ctrl++` / `Ctrl+-` で webview の `setZoomFactor` を操作。
  小さいウィンドウで読みやすく。

- [ ] **ページをピン留め（起動時に自動ロード）**
  特定 URL を「固定ページ」として設定。アプリ起動と同時に読み込む。

---

## 優先度: 低（あると面白いがニッチ）

- [ ] **スクリーンショットボタン**
  FloatPad ウィンドウ全体または webview のスクリーンショットを PNG として保存。

- [ ] **カスタム CSS インジェクション（webview）**
  設定画面で CSS を書くと、すべての webview ページに適用。
  広告ブロックや文字サイズ変更に使える。

- [ ] **ポモドーロタイマータブ**
  25分集中 / 5分休憩のシンプルなタイマーを新しいタブとして追加。
  OS 通知と連携。

- [ ] **カラーピッカーサブ機能（電卓タブ内）**
  HEX / RGB / HSL の相互変換ツールを電卓タブに小さく追加。
  デザイン作業中に重宝。

- [ ] **クリップボード履歴（ミニ）**
  直近 5〜10 件のコピー履歴をドロップダウン表示。
  ウィジェットとして常に浮かんでいる FloatPad の強みを活かせる。

- [ ] **カスタムタブ名・アイコン変更**
  設定パネルで各タブのラベルや絵文字を変更できるようにする。

- [ ] **アクセントカラーのカスタム入力（HEX 直打ち）**
  現状は 5 色のスウォッチのみ。任意の HEX カラーを入力できる欄を追加。

## 技術的な改善（機能追加ではないが価値あり）

- [ ] **IPC 型定義の共有**
  `preload/index.ts` の `api` オブジェクト型を renderer 側でも型安全に使えるよう
  共通の型定義ファイル（`src/types/api.d.ts`）として切り出す。

- [ ] **自動アップデート（electron-updater）**
  GitHub Releases からアップデートを自動検出・ダウンロード。

- [ ] **エラーバウンダリ / クラッシュレポート**
  renderer のエラーをキャッチして「再起動」ダイアログを表示する仕組み。

- [ ] **E2E テストの拡充**
  現在の `scripts/verify.ts` を Jest + Playwright に移行し、
  CI（GitHub Actions）で自動実行。
